# vue3-elementPlus-vite

> 这是一个vue3 的后台模版，主要目的是为了让广大开发者，能够快速上手即用的，所以页面我就没写太多，大家可以按照自己公司的项目来进行删减

### 主要依赖

- vue3 
- elementPlus
- vite

### 主要封装

- 侧边栏
- 路由权限
- 登录操作
- 路由记录
- 按需引入
